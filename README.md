# Stock Price Prediction App

A Streamlit-based web application for predicting stock prices using machine learning.

## Features

- Real-time stock price visualization
- Price prediction using machine learning models
- Historical data analysis
- Interactive charts and technical indicators

## Demo

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://stock-price-prediction123.streamlit.app/)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/prem-pur/-Stock-Price-Prediction.git
cd stock-price-prediction
```

2. Create and activate a virtual environment:
```bash
python -m venv env
.\env\Scripts\activate  # On Windows
```

3. Install the required packages:
```bash
pip install -r requirements.txt
```

## Usage

1. Run the Streamlit app:
```bash
streamlit run app.py
```

2. Open your browser and navigate to `http://localhost:8501`
